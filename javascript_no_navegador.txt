Podemos execultar javascript em um navegador

Podemos utilizar a tag script em documento HTML

Hoje todo navegador moderno possui o console para javascript

Um console de navegador utilizar REPL um ambiente de programacao de computador interativo e simples que recebe entradas de um unico usuario as execulta e retorna o resultado

Um REPL tambem pode ser chamado de interactive toplevel ou language shell

Um REPL permite execultar javascript dentro do contexto da pagina

Podemos modificar o DOM ou fazer login no console

No console podemos visualizar mensagens que foram registradas pelo codigo ou qualquer outra informacao com solicitacoes de rede e erros de seguranca

Math.sqrt() => funcao que retorna a raiz quadrada de um numero
